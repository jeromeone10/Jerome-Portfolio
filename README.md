# Jerome M. Niñal - Portfolio

A modern, responsive personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Features

- **Responsive Design**: Optimized for all screen sizes (mobile, tablet, desktop).
- **Dark Mode**: Default sleek dark UI for a professional look.
- **Smooth Scrolling**: Easy navigation between sections.
- **Animations**: Subtle and modern animations using Framer Motion.
- **Project Showcase**: Highlights key projects with tech stacks and links.
- **Skills Categorization**: Organized technical skills for easy reading.
- **Contact Form**: Functional UI for project inquiries.

## Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Navigation**: React Scroll

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd "Jerome Portfolio"
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173`.
